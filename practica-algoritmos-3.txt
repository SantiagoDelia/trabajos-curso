---------------------------------------
problematica:
El usuario quiere saber si puede entrar a una discotecta segun la edad del cliente
---------------------------------------

---------------------------------------
Algoritmo:
1. Solicitar al usuario que ingrese la edad del cliente  
2. Verificar si la edad ingresada cumple con el requisito para ingresar a la discotecta
3. Mostrar al usuario si su cliente puede o no ingresar a la discoteca
---------------------------------------

---------------------------------------
Pseudocodigo:

Inicio
    #Paso 1: Solicitar al usuario que ingrese la edad del cliente
      Mostrar mensaje: "Por favor, ingrese la edad del cliente: "
      Leer el dato ingresado y asignarlo a la variable edad 

    #Paso 2: Verificar si la edad ingresada cumple con el requisito para ingresar a la discotecta
      Si edad >= 18 entonces
      asignarle a la variable permitido que sea verdadero 
      Sino 
      Asignarle a la variable permitido que sea falso

    #Paso 3: Mostrar al usuario si su cliente puede o no ingresar a la discoteca
    Si permitido es true 
    Mostrar mensaje "El cliente puede ingresar"
    Sino 
    Mostrar mensaje "Lo sentimos, no puedes ingresar siendo menor de edad"
    Fin si
Fin
